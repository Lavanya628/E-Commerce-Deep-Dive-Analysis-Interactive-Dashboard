# E-Commerce-Deep-Dive-Analysis-Interactive-Dashboard #
This project focuses on performing a deep-dive analysis of an e-commerce dataset and building an interactive Power BI dashboard to extract meaningful business insights.

# Objective
To analyze complex business data, define key performance indicators (KPIs), and create an interactive dashboard for decision-making.

# Tools & Technologies
Python (Pandas, NumPy)
Power BI
Data Cleaning & Wrangling
Data Visualization

# Key KPIs
Total Sales
Total Profit
Total Orders
Average Order Value (AOV)
Profit Margin %

# Analysis Performed
Sales and profit analysis by category and region
Monthly sales trend analysis
Top customer identification
Loss-making category analysis

# Dashboard Features
Interactive filters (Region, Category, Month)
KPI summary cards
Dynamic charts and visuals
Conditional formatting for profit/loss

# Key Insights
Electronics category generates the highest revenue
Grocery category is loss-making
Regional performance varies significantly
Top customers contribute a large share of sales
